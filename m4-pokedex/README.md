# M4 — Pokédex (Bootstrap + JS + Fetch)

## Descripción
Pokédex de 9 Pokémon (ID 1–9) consumiendo PokeAPI, renderizada con cards responsive.
Diseño mejorado con Glassmorphism y animaciones suaves.

## Tecnologías
- HTML, CSS (Custom + Bootstrap 5)
- JavaScript ES6+, Fetch API, async/await

## Ejecutar
- Abrir `index.html` en el navegador (o Live Server).

## Características
- **Fetch en paralelo** de los primeros 9 Pokémon.
- **Búsqueda en tiempo real** por nombre.
- **Diseño Responsive** y "Premium" con efectos visuales.
